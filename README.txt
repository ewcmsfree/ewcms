企业网站信息内容管理平台（EWCMS）
目标

    简单方便的完成，用户站点管理和生成。
    通过ftp、sftp等手段，在线发布和管理用户托管网站。
    通过Webservic技术，实现于其他业务系统的集成。
    引入SEO功能，辅助用户对站点在搜索引擎中的优化，主要针对(google和baidu)。 

技术路线

    JDK：6.0以上。
    开发架构：J2EE。
    操作系统：兼容全部主流服务操作系统，包括Windows Server、 Linux。
    数据库：Postgresql 8.0以上。
    应用服务器：Tomcat6.0以上。
